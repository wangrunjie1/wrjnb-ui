按钮 Button

按钮用于触发一个操作或事件，如提交表单、取消操作等。

基础用法

示例代码

<!-- 主要按钮 -->
<button class="vc-button vc-button--primary">主要按钮</button>

<!-- 次要按钮 -->
<button class="vc-button vc-button--secondary">次要按钮</button>

<!-- 危险按钮 -->
<button class="vc-button vc-button--danger">危险操作</button>

Vue 组件代码

// Button.vue
<template>
  <button 
    :class="[
      'vc-button',
      `vc-button--${type}`,
      `vc-button--${size}`,
      { 'vc-button--disabled': disabled }
    ]"
    :disabled="disabled"
    @click="handleClick"
  >
    <slot></slot>
  </button>
</template>

<script setup>
defineProps({
  type: {
    type: String,
    default: 'primary',
    validator: (value) => {
      return ['primary', 'secondary', 'danger', 'text'].includes(value)
    }
  },
  size: {
    type: String,
    default: 'medium',
    validator: (value) => {
      return ['small', 'medium', 'large'].includes(value)
    }
  },
  disabled: Boolean
})

const emit = defineEmits(['click'])

function handleClick(event) {
  emit('click', event)
}
</script>

属性说明

属性名 说明 类型 默认值
type 按钮类型 'primary' | 'secondary' | 'danger' | 'text' 'primary'
size 按钮尺寸 'small' | 'medium' | 'large' 'medium'
disabled 是否禁用状态 Boolean false

事件说明

事件名 说明 回调参数
click 点击按钮时触发 event: MouseEvent
